HTML

  • The head element contains information about the webpage.
  • The metadata in the head element informs the browser how to load the page.
  • Elements in the head element are not visible to the user.
  • The body element represents the visible content shown to the user.
  • Try to only use semantic elements in the body.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

  • git commit -m "add message text here" : commits to repo with -m flagging the message accompanying the commit
